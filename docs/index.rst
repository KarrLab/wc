wc documentation
================
Integrated user interface for whole-cell models and whole-cell modeling tools

The integrated user interface provides modelers the following capabilities:

* Display the available models and tools
* Display the available operations for each model and tool 
* Run whole-cell models and whole-cell modeling tools
